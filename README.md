# Web Ucapan Ulang Tahun

Website ucapan ulang tahun interaktif dengan 3 pilihan tema.

## Fitur
- Intro screen dengan animasi
- Countdown real-time ke hari ulang tahun
- Amplop pesan yang bisa dibuka
- Musik latar
- 3 tema: Romantic Rose Gold, Colorful & Fun, Elegant Minimalist

## Tema
1. **Template A - Romantic Rose Gold**: [https://hbd-web-two.vercel.app/template-A/]
2. **Template B - Colorful & Fun**: [https://hbd-web-two.vercel.app/template-B/]
3. **Template C - Elegant Minimalist**: [https://hbd-web-two.vercel.app/template-C/]

## Tech Stack
HTML, CSS, JavaScript (vanilla, no framework)

## Cara Pakai
1. Pilih tema
2. Edit nama, tanggal, foto, dan pesan di file HTML
3. Ganti file audio di folder `audio/`
